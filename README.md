# Library Management System Database

## Overview

This project contains a complete Library Management System database designed for universities, colleges, and educational institutions. The database is developed in Microsoft SQL Server (SSMS) and includes all major modules required for managing library operations efficiently.

The system supports library branches, members, books, circulation, acquisitions, fines, employees, room management, and organizational structure.

---

## Features

### Library Management

* Multiple library branches
* Library sections and management units
* Library racks and shelves
* Branch location management

### Member Management

* Library member registration
* Membership cards
* Member blocking system
* Custom borrowing quotas

### Book & Item Catalog

* Book and item information
* Authors and publishers
* Subject headings
* Categories and editions
* Language management
* Item copies and accession numbers

### Circulation Management

* Book issuance
* Return management
* Reservation system
* Borrowing rules
* Loan duration management

### Fine Management

* Fine categories
* Fine definitions
* User fines
* Fine transactions and payments

### Acquisition & Purchasing

* Acquisition records
* Item quotations
* Purchase management
* Vendor and supplier records

### Employee Management

* Employee information
* Designations
* Department assignments
* Employee relation history

### Room & Facility Management

* Room management
* Room types
* Feature allocation
* Facility tracking

### Organizational Structure

* University hierarchy
* Campuses
* Faculties
* Institutes
* Departments
* Structure units and designations

## Installation

1. Open Microsoft SQL Server Management Studio (SSMS).
2. Create a new database.
3. Open the provided SQL script file.
4. Execute the script using **F5**.
5. All tables, keys, and relationships will be created automatically.

## Project Purpose

The purpose of this project is to provide a complete database solution for managing library operations within educational institutions. The design focuses on scalability, data integrity, and efficient management of library resources.
